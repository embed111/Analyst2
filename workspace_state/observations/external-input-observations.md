# 外部输入观察日志（非用户偏好）

- 最后更新: 2026-02-25
- 目的: 记录训练师或其他外部角色的偏好/指令风格，避免混入 `user_profile/`。

## 记录规则
1. 必须标注来源角色（如 `trainer`、`reviewer`、`simulator`）。
2. 必须标注归属状态：
   - `external_confirmed`: 明确属于外部角色。
   - `pending_attribution`: 来源不明确，待归属。
3. 外部偏好默认不写入 `user_profile/`。
4. 仅当用户本人明确确认“这是我的长期偏好”时，才允许迁移到 `user_profile/` 并在变更日志说明迁移理由。
